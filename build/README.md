# Build Assets

Bu klasör uygulama ikonlarını içerir.

## Gerekli Dosyalar

Windows için:
- `icon.ico` - 256x256px icon dosyası

macOS için:
- `icon.icns` - macOS icon paketi

Linux için:
- `icon.png` - 512x512px PNG icon

## Icon Oluşturma

Şimdilik varsayılan Electron ikonu kullanılacaktır. Özel icon eklemek için:

1. 512x512px PNG dosyası hazırlayın
2. Online araçlarla .ico ve .icns formatına çevirin
3. Bu klasöre ekleyin

Önerilen araçlar:
- https://www.icoconverter.com/
- https://cloudconvert.com/
